Css - 利用伪元素的 content 属性实现 tooltip 插件的功能

  当要实现元素 hover 的时候显示提示信息，如果用 title 属性觉得效果太弱，但
是又不想用JS写，更不想引入一个 tooltip 第三方的大库，这个时候可以用CSS3的
attr 属性实现，将想要展示的提示内容放到一个属性里面，而且样式可以由自己设计！
